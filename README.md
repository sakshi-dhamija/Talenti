# Talenti: Interview Creation Portal

A simple web app where admin can schedule interviews by selecting participants, start time and end time of the interview.  


## Tech Stack
<ol>
<li>HTML</li>
<li>CSS</li>
<li>JavaScript</li>
<li>NodeJs</li>
<li>ExpressJs</li>
<li>MySQL</li>
</ol>
